# Maze
강화학습을 통하여 에이전트 학습시키기 - 미로 통과

- 6x6 미로 환경
- 10개의 벽이 있다고 가정 (벽의 위치는 임의로 지정)
- 시작점은 왼쪽 위, 도착점은 오른쪽 밑
- 행동가치함수 기반의 행동 정책 결정
- 에이전트는 화살표가 가르키는 곳으로 이동
- 화살표가 사방을 향하는 곳은 벽
- max step 최대 10 안으로 도달하는 것이 목표
- max step = 5에 목적지에 도착 -> 목표 달성
